## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Backend integration or data layer | Site remains static/hardcoded |
| Real team/jobsite photos | Owner hasn't provided them yet |
| Downloadable PDF prequal package | Goes stale, worse than no PDF |
| Animated safety stat counters | Trivializes serious data |
| OSHA 300 log display | Legal exposure, not industry standard |
| Google Maps embed | Adds external dependency, loading overhead for minimal value |
