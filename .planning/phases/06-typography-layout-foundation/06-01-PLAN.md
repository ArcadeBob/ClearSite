1. Add variant type and update props interface:
   ```typescript
   type SectionHeaderVariant = 'left-bar' | 'banner' | 'overlapping';
   ```
   Add `variant?: SectionHeaderVariant` to SectionHeaderProps. Default to `'left-bar'`.

2. Implement three variant layouts (all headings use `font-heading uppercase tracking-wide`):

   **left-bar**: Vertical orange accent bar on left edge, text left-aligned.
   - Outer div: `border-l-4 border-accent pl-6` (replaces `text-center`)
   - Subheading: keep, left-aligned, accent color
   - Title h2: `font-heading text-brand uppercase tracking-wide` + size classes
   - Description: `max-w-2xl` (no `mx-auto` since left-aligned)

   **banner**: Full-width dark navy background, white text, centered.
   - Outer div: `bg-brand py-10 px-6 -mx-4 sm:-mx-6 lg:-mx-8` for break-out effect
   - Inner wrapper: `max-w-7xl mx-auto text-center`
   - Subheading: `text-accent` (orange on dark navy for contrast)
   - Title h2: `font-heading text-white uppercase tracking-wide` + size classes
   - Description: `text-slate-300 max-w-2xl mx-auto`

   **overlapping**: Large ghost title behind real title.
   - Outer div: `relative`
   - Ghost h2: same title text, `font-heading text-5xl md:text-7xl text-brand/10 uppercase tracking-wide absolute -top-8 left-0 select-none pointer-events-none` with `aria-hidden="true"`
   - Real content in `relative pt-6` wrapper
   - Subheading: omit (do not render) -- the oversized ghost IS the visual statement
   - Title h2: `font-heading text-brand uppercase tracking-wide` + size classes
   - Description: `max-w-2xl`

3. Keep `titleSize` logic working across all variants. For `titleSize === 'sm'`: use `text-2xl md:text-3xl`, for `'lg'`: use `text-3xl md:text-4xl`.

4. Keep `className` prop working -- append it to the outermost div of each variant.

5. Keep `subheadingColor` prop working for left-bar and banner variants (overlapping drops the subheading entirely).

IMPORTANT: Do NOT use `text-slate-900` on any heading -- use `text-brand` (or `text-white` for banner variant). Per user decision, heading color changes from slate to brand navy.
